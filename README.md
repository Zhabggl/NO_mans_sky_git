# 无人深空博物馆

一个展示《无人深空》游戏中各种文物、标本和遗迹的在线博物馆。

## 网站地址

访问: https://[你的用户名].github.io/NO_mans_sky_git

## 功能特点

- 文物分类浏览（阿特拉斯文物、标本类、技术装置、生物文物等）
- 智能搜索功能
- 文物详情展示
- 科幻风格星空背景
- 响应式设计，支持移动端

## 文物分类

- **阿特拉斯文物** - 与阿特拉斯相关的神秘文物
- **标本类文物** - 各类生物和植物标本
- **技术装置** - 先进的科技设备
- **生物文物** - 各种奇特的生物样本
- **生物标本** - 详细的生物标本记录
- **文献资料** - 古老的文字和记录
- **遗物类文物** - 古代文明的遗留物品
- **植物文物** - 珍稀的植物样本
- **植物标本** - 详细的植物标本记录

## 本地运行

```bash
# 进入项目目录
cd NO_mans_sky_git

# 启动本地服务器
python -m http.server 8000

# 访问 http://localhost:8000
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 纯前端实现，无需后端

## 数据来源

文物数据来源于《无人深空》游戏内的文物清单。
